# EZQL

EZQL is rust ORM :rocket:.
My goal is to make it easy to use and easy to understand :smile: without need to learn SQL :muscle:.

## Features

-   Queries
    -   [x] Select
    -   [x] Insert
    -   [ ] Update
    -   [ ] Delete
-   Relations
    -   [ ] One to One
    -   [ ] One to Many
    -   [ ] Many to Many
-   Migrations
    -   [ ] Create
    -   [ ] Drop
    -   [ ] Alter
-   SQL Dialects
    -   [ ] MySQL
    -   [ ] PostgreSQL
    -   [x] SQLite
    -   [ ] SQL Server
